Center 统一调度，Worker 在任意机器运行，Agent 数字员工承接每一次派单 —— 你与他们在同一条可追溯的线程上协作。

CENTER Center server · Web Console · SQLite · Supervisor
派单 · 控制流 ↓   ↑ enroll · 心跳
WORKER ×N Worker 任意机器 · daemon
拉起 · 监督 ↓
AGENT ×N Agent 数字员工 claude / codex / opencode
MCP 直连 Center ↗ 发消息 · 建任务 · 请求输入 —— 每一步都落在 可追溯线程 上。
