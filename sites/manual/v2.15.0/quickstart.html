5 分钟快速开始

从零到第一个 Task 完成 —— 六步走完。

Prerequisites

  • Go 1.22+（从源码构建时需要；使用发行包则不需要）
  • macOS 或 Linux（Windows 未经测试）
  • 磁盘空间 100 MB 以上（SQLite + 二进制 + 日志）

Step 1 — 安装 center

方式 A：从源码构建

git clone https://github.com/oopslink/agent-center.git
cd agent-center
make build          # 构建 frontend (vite) + backend (Go) → ./bin/agent-center

方式 B：使用发行包

# 下载对应平台的 release tarball
tar xzf agent-center-v2.15.0-darwin-arm64.tar.gz
cd agent-center-v2.15.0-darwin-arm64/
./install center    # 幂等，检测已有安装会跳过

方式 C：source guided installer

curl -fsSL https://raw.githubusercontent.com/oopslink/agent-center/main/install.sh | bash
install center 默认前台模式，仅部署文件与配置。如需后台服务（launchd / systemd），加 --service。

Step 2 — 启动 center

agent-center server --config=~/.agent-center/etc/config.yaml

启动后终端会输出：

agent-center server: db=... listen=:7050 web=127.0.0.1:7100 admin=...
默认端口：Web Console 127.0.0.1:7100，server :7050，Admin TCP 0.0.0.0:7300（worker 连接用）。

Step 3 — 打开 Web Console

浏览器访问：

http://localhost:7100

首次打开会 mint bootstrap admin token。你将看到 agent-center SPA：Channels 列表、Fleet 页面、Task 面板。

Web Console 仅绑定 loopback（127.0.0.1）。远程访问请用 SSH tunnel：ssh -L 7100:127.0.0.1:7100 user@host

Step 4 — 安装 worker

在 Web Console 点击 "+ Add Worker"，输入名称，复制生成的安装命令（含 bootstrap URL、enroll token、server fingerprint）：

./install worker \
  --bootstrap=tcp://HOST:7300 \
  --server-fingerprint=sha256:... \
  --token=enroll_... \
  --worker-id=worker-... --worker-name="my box"

Worker 可以与 center 同机，也可以在另一台开发机上。安装后启动：

agent-center worker run --config=~/.agent-center/workers/worker-.../etc/config.yaml
同样默认前台模式。加 --service 可注册 LaunchAgent (macOS) 或 systemd unit (Linux)。

Step 5 — Enroll worker

Worker 启动时会自动使用 --token 完成 enroll 握手。成功后终端输出：

[worker] enrolled as worker_id=worker-...

在 Web Console 的 Fleet 页面可以看到 worker 状态变为 online。

Step 6 — 在 Web Console 创建第一个 Task

  1. 进入 Channels → 创建一个 channel
  2. 发一条消息
  3. 点击消息 → "Derive Task" → 选择项目 → 确认
  4. Task 出现在 Tasks 面板；状态自动流转：submitted → working → completed
  5. 在 Fleet 页面查看 worker 的执行计数
也可以通过 CLI 创建 Task：agent-center task new --project-id=demo --title="hello world"

更详细的安装与部署说明见 安装指南。
