# 开发周期编排模板（Cycle Flow Template）— 指针

> ⚠️ **没有内置模版了**（2026-07 复盘决定：模版全是用户自管 / UI 可编辑的，无 builtin 类型）。
> 原来的内置 cycle 模版已删除，其 **9 步全真验收开发流程**内容搬进用户模版 **「全真验收开发模式」**。
>
> - **看/改模版**：用 `list_templates` / `get_template` / `update_template` 等 MCP 工具，或在 Web Console 的 Templates 页 Edit。
> - **建 plan 时**：用「全真验收开发模式」这条模版 scaffold（不再有 `get_template("cycle")` 的内置 cycle）。
>
> 全真验收怎么做（起隔离实例、逐项验、截图证据）见 `docs/rules/acceptance-methodology.md` + `docs/rules/acceptance-checklist.md`。
