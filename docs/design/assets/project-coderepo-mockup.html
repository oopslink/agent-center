设计要点（详见 docs/design/features/project-coderepo.md）

  • 多仓:每项目 N 个仓,主仓(primary)用于 Integrate merge-check;每仓 {label,url,provider,default_branch,primary} + 只读凭据。
  • agent 接口:MCP list_project_repos / get_repo_info 返回静态配置(便宜免凭据);get_repo_info(live=true) 可选返回 remote 最近 commits/branches。
  • remote 查看:provider 抽象,v1 GitHub 走 go-github;未知 provider 回退 git ls-remote。不 clone。
  • 凭据:项目级只读 token,加密落地;供 viewing + agent live 查询。
