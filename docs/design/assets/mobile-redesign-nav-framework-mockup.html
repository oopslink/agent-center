关键规则

  • 底部 Tab 固定 5 个模块（Work / Chat / Team / Remind / Sys），与 PC col① rail 一一对应，图标+短标签。
  • 顶部栏左侧 = 当前模块名 + ☰，点击打开二级导航 BottomSheet（列出该模块子项，如 Projects/Issues/Tasks/Plan）。
  • 顶部栏右侧 = 🔔 attention 徽标（未读数）+ 👤 账户入口，行为不变（沿用现有 mobile top bar 语义）。
  • 详情类页面（Task/Project/Issue/Plan Detail）顶部栏左侧变为 ‹ 返回，Context Panel 通过页面内的"更多信息"按钮触发 BottomSheet，而非常驻。
  • 所有 BottomSheet 支持下拉手势关闭 + 点击遮罩关闭，最大高度不超过 70vh，避免遮住返回路径。
