# v2.6 Development Plan（Phase 拆分）

| Field | Value |
|---|---|
| Owner | @AgentCenterPD（PM）启稿 → 跟 @oopslink review → 跟 @AgentCenterDev align 后 ratify |

## § 6. 并行机会

Phase 1 内部：
- BE-1 是 hard prereq for BE-2/3/4
- BE-5 可与 BE-2/3/4 并行（events 类型定义独立）

Phase 1 ↔ Phase 2：
- BE-6/8 等 BE-1 ready 即可开
- BE-7/9 等 BE-1+BE-4 ready 即可开

- FE-1 等 BE-2
- FE-2/3/4 等 BE-2/3/4
- FE-5/6 等 BE-2 + BE-7

## § 7. 风险评估

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| DS-2 race 实测失败 | 中 | 严重（数据 corruption）| BE-3 加并发 test；OrganizationLockManager 充分测试 |
| SQLite FK constraint 未启用 | 中 | 中（约束失效）| BE-1 启动时 `PRAGMA foreign_keys=ON;` |
| JWT cookie 跨域问题 | 低 | 中 | localhost 单域；生产 https |
| Supervisor 砍除影响现有用户 | 低 | 中 | release note 说明替代路径；自定义 admin agent |
| 现有 SPA bundle 大改动 | 中 | 中 | FE-2/FE-6 一起 refactor，避免分阶段 |
| Identity Migration 顺序错 | 低 | 严重 | 一次 drop-and-recreate，无 partial migration |
