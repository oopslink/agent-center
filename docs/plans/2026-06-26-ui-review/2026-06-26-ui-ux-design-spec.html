MC-1 P1
Composer Send/Attach 按钮 32px
Send 和 Attach 按钮 h-8 w-8 = 32px，低于移动端 44px 最低标准。Send 是移动端最高频操作。在 @media (pointer: coarse) 下提升至 h-11 w-11 (44px)。

ThreadSidebar 移动端 min/max 矛盾
minWidth=320px > 75vw=281px (375px screen)。移动端改为全屏 inset-0，隐藏 ResizeHandle，back arrow 替代 X close。

ConversationView loading 无语义角色
Loading 用 <p> 纯文本，缺少 role="status"；Error 缺少 role="alert"。添加骨架屏替代纯文本，改善移动端视觉。

Global Standard

Touch Target Utility Class

建议在 Tailwind config 或 index.css 中定义全局 touch-target utility，所有交互元素在触控设备上自动满足 44px 标准。

/* index.css — global touch target utility */
@media (pointer: coarse) {
  button,
  [role="tab"],
  [role="menuitem"],
  .touch-target {
    min-height: 44px;
    min-width: 44px;
  }
}

/* tailwind.config.js — custom utility */
theme.extend.minHeight: {
  'touch': '44px',
}
theme.extend.minWidth: {
  'touch': '44px',
}
