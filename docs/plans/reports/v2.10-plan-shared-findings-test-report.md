## 测试计划

### 范围
- **被测**：plan-scoped 共享 findings 的写/读/retract + 入库 admission + 派单注入 + agent 工具面 + 迁移。
- **不在范围**（推迟，见 ADR-0053 §Consequences）：LLM-verifier、分层 unfold、FE 查看 UI、失败自动落 gist、built-in pool 注入。

### 前置
- 环境：本地，Go 1.25，in-memory SQLite（`persistence.MemoryDSN()`），httptest in-process admin server，真 `bin/agent-center`（deployed-smoke）。
- 数据：每用例自建（无共享可变 fixture）。
