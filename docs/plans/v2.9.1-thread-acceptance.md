## 1. 特性范围

消息 **Thread**：从任一消息派生 thread、thread 内继续发消息，**Slack 式 1 层**（回复扁平、不嵌套），**所有会话类型**（channel / DM / issue / task / plan）通用。Owner 2026-06-13 拍板：Slack 1 层 / 全会话 / 先做"有无新活动" / @agent-in-thread 唤醒=是。
